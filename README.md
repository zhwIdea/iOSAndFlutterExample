# iOSAndFlutterExample
基于FlutterBoost插件，实现iOS与Flutter混合开发和通信，Flutter与原生页面相互跳转并传参，干货满满。               
项目文档：https://www.jianshu.com/p/3b320a48a80f                                                                                    
下载demo后，需在iOS工程中，执行pod install命令，项目才可正常运行。
